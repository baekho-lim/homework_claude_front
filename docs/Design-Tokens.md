# Design Tokens (hw) v0.1

## CSS Variables (names are invariant)
- --hw-bg: app background
- --hw-fg: primary text
- --hw-muted: muted text / surfaces
- --hw-accent: primary action
- --hw-border: border color
- --hw-radius-lg: large radius
- --hw-space-1/2/3: spacing scale
- --hw-font-sans: base font family

## Rule
- 새로운 스타일 실험(AI Studio)은 토큰 "값"은 바꿔도 되지만,
  토큰 "이름"은 절대 바꾸지 않는다.
