# Wireframe (Google AI Studio)

## 용도
Google AI Studio에서 만든 실험적 코드를 다운로드하는 공간

## 상태
- **동적**: 계속 변경됨
- **실험용**: 아직 확정되지 않은 기획/디자인

## 워크플로우
```
Google AI Studio (실험)
        ↓
    다운로드 → 이 폴더에 덮어쓰기
        ↓
    기획 검토 & 수정
        ↓
    버전 확정 시 → docs/Diff-Template.md 작성
        ↓
    IDE에서 실제 개발 (src/)
```

## 주의
- 이 폴더는 **작업 공간**입니다
- 확정된 코드는 `src/`로 이동합니다
- 버전이 찍히면 브랜치로 보관합니다
